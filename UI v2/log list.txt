Выравнивание кнопок:

Изменен код для создания кнопок с использованием функции CreateButton, чтобы избежать дублирования кода.
Каждой кнопке назначены координаты, чтобы обеспечить более равномерное распределение на форме.
Упрощение кода:

Убраны ненужные элементы, такие как ползунок для масштаба, чтобы упростить код и избежать путаницы.
Добавление координатных осей:

Добавлена функция DrawAxes для рисования горизонтальной и вертикальной координатных осей.
Улучшение функции рисования графика:

Использованы более понятные переменные scaleX и scaleY для определения масштаба графика.
Вместо жесткой привязки к значениям ширины и высоты используется plotPictureBox.Width и plotPictureBox.Height для более гибкого кода.
Обновленный логику рисования графика:

Изменен метод DrawFunction для использования функции DrawAxes и общей логики рисования.
Убраны ненужные переменные и параметры для повышения читаемости.
Изменение метода создания графического окна:

Изменено имя класса на SimpleGraphicalCalculator для лучшего соответствия характеру приложения.
Изменено имя метода MainForm на InitializeUI, чтобы лучше отражать его назначение.
Улучшение читаемости:

Использованы более понятные имена переменных и методов для улучшения читаемости кода.
Уточнение комментариев:

Добавлены комментарии для пояснения отдельных частей кода.
