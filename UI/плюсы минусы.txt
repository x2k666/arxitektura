Плюсы:

Интерактивность: Пользователь может выбирать различные функции для построения графиков, что делает приложение интересным и визуально привлекательным.

Визуализация данных: Графики позволяют пользователям лучше понимать характер выбранных функций, что полезно для образовательных целей.

Простота использования: Интерфейс приложения прост и понятен, что делает его удобным в использовании для широкого круга пользователей.

Динамическое обновление: При изменении функции пользователь может моментально увидеть изменения на графике, что обеспечивает динамичный опыт.

Минусы:
Некрасивое оформление. Кнопки не выровнены.

Отсутствие координатной сетки: Отсутствие координатной сетки может затруднить точное определение значений функции на графике.

Ограниченные параметры функций: Пользователь не может настраивать параметры функций (например, изменять масштаб по осям), что может быть полезным для более детального изучения графиков.

Ограниченная точность: Простая реализация может не обеспечивать высокой точности при построении графиков, что может быть проблемой для некоторых приложений.

Отсутствие сохранения данных: Нет функционала сохранения или загрузки графиков для последующего анализа, что может быть полезным для пользователей, требующих сохранение результатов.

Недостаточная информация: Пользователь может испытывать затруднения в определении значений функции в конкретных точках, так как не предоставляется информация о координатах.
